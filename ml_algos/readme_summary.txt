Machine Learning Algorithms - Interview Focused Summary

This file keeps only the most useful and commonly asked machine learning algorithms in interviews.

Supervised Learning

1. Linear Regression - Predicts a numeric value using a straight line relationship.
2. Logistic Regression - Predicts a class or yes/no output.
3. Decision Tree - Splits data using simple question-based rules.
4. Random Forest - Combines many decision trees for better prediction.
5. Support Vector Machine (SVM) - Separates classes using the best boundary.
6. K-Nearest Neighbors (KNN) - Predicts based on the nearest similar data points.
7. Naive Bayes - Uses probability to classify data quickly.
8. Ridge Regression - Linear regression with regularization to reduce overfitting.
9. Lasso Regression - Linear regression that can also remove less important features.

Ensemble Learning

10. Bagging - Trains multiple models on random samples and combines them.
11. AdaBoost - Improves prediction by focusing on previous mistakes.
12. Gradient Boosting - Builds models step by step to reduce errors.
13. XGBoost - A popular and powerful boosting algorithm used in interviews and competitions.

Unsupervised Learning

14. K-Means - Groups similar data into clusters.
15. Hierarchical Clustering - Creates clusters in a tree-like structure.
16. DBSCAN - Finds dense clusters and detects noise points.
17. Principal Component Analysis (PCA) - Reduces features while keeping important information.

Quick Remember
- Regression algorithms predict numbers.
- Classification algorithms predict categories.
- Clustering algorithms group similar data.
- PCA reduces the number of features.
- Neural networks learn complex patterns.
